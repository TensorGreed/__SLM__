# Glossary

## SLM
Small Language Model tuned for a constrained domain or task.

## Domain Pack
Pack-level defaults and overlays for domain workflows.

## Domain Profile
Contract describing task, compliance, and runtime preferences.

## Starter Pack
Novice-oriented domain bootstrap defaults (model family, adapter, gates, target, safety reminders).

## Target Profile
Deployment environment profile (mobile, edge, server, browser) with constraints.

## Gate Policy
Rules that determine whether a model can move to next stage/deployment.

## Measured Metric
Metric collected from a real benchmark run.

## Estimated Metric
Fallback metric derived from heuristics or prior calibrations.

## Strict Mode
Execution mode that avoids hidden fallback behavior and surfaces hard blockers explicitly.

## Remediation Plan
Actionable set of data/training changes derived from evaluation failures.
